# Pokedex API Demonstration
### Overview
This project is a demonstration of API access. It uses two python modules to:
-   run a user interface
-   use functions to read and retrieve information from an API.

It acts as a demonstration of acccessing and displaying data from an API, and can be used as further reference. It's main use is to help users learn information about different pokemon.
